Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica).

Nessa frase temos um termo em destaque usando a tag STRONG (semântica).

Itálico / ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

Podemos também criar um texto marcado utilizando a tag mark.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado depois.

Támbém podemos marcar o texto como sublinhado com a tag U (não semântica).

Texto sobrescrito

Para inserir coisas como x10 + 69 utilizamos a tag sup.

Texto subescrito

Para inserir coisas como H2O utilizamos a tag sub.
